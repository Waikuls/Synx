# Project Structure
- `src/source.luau` is the core UI library.
- `Fatality/main.lua` is the recommended main entry script for normal use.
- `examples/main.luau` is a full example script.
- `examples/config-manager.luau` is a smaller example showing config usage.
- `example.luau` is a legacy root example kept for compatibility.
