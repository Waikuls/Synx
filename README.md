# Usage
- [Document](https://cat-sus.gitbook.io/fatality)
- [Example](https://github.com/4lpaca-pin/Fatality/blob/main/example.luau)
- For GitHub/raw execution, prefer the repo-root `main.lua` bootstrap so it can cache-bust `Fatality/main.lua` automatically.
- `Fatality/main.lua` remains the internal app entry and local/dev entry.

# Project Structure
- `src/source.luau` is the core UI library.
- `main.lua` is the recommended remote/bootstrap entry script.
- `Fatality/main.lua` is the internal app entry and the local/dev entry script.
- `examples/main.luau` is a full example script.
- `examples/config-manager.luau` is a smaller example showing config usage.
- `example.luau` is a legacy root example kept for compatibility.
